# Project: Tasks management system
----
## A REST-API for Tasks Management System written in Golang.
----
# Require:
```bash
docker
git 
```

# How to install:
```bash
git clone https://github.com/leehai1107/Task-managent-sys
```
# How to run:
```bash
docker compose up
```

# Project structure
